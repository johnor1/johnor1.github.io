Projektit

1. Digitaalinen ajanvarausjärjestelmä (Kuvitteellinen)

Suunnittelin ja toteutin prototyypin ajanvarausjärjestelmästä pienyritykselle. Käytin HTML:ää, CSS:ää ja JavaScriptiä. Käyttäjäpolkuja testattiin heuristisen arvioinnin avulla.

2. Responsiivinen verkkosivupohja (Kuvitteellinen)

Rakensin mobiiliystävällisen verkkosivupohjan käyttäen Flexboxia ja CSS Grid -tekniikoita. Sivupohja on suunniteltu portfolio- ja CV-käyttöön.

3. UX-auditointi: Karkkilan kaupungin verkkosivut (Kuvitteellinen)

Analysoin kaupungin verkkosivujen käytettävyyttä ja laadin kehitysehdotuksia. Käytin palvelumuotoilun menetelmiä, kuten käyttäjäpersoonia ja asiakaspolkuja.

4. GitHub README-optimointi (Kuvitteellinen)

Harjoittelin versionhallintaa ja dokumentointia GitHubissa. Kirjoitin selkeän README-tiedoston, jossa esittelin projektin rakenteen, teknologiat ja käyttöohjeet.
